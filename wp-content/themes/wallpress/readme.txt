WallPress Theme
http://www.designwall.com/themes/wallpress

-----------------------------------------------
    =INSTALL=
-----------------------------------------------

1. Upload the WallPress theme folder via FTP to your wp-content/themes/ directory.
2. Go to your WordPress dashboard and select Appearance.
3. Activate the WallPress theme.
